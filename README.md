# 🎮 Pong Game (HTML, CSS, JavaScript)

A modern and interactive **Pong game** built using pure **HTML, CSS, and JavaScript**, featuring smooth gameplay, intelligent AI, and realistic physics.

---

## 🚀 Features

* 🕹️ Smooth paddle control using **mouse or arrow keys**
* 🤖 **Computer AI** that dynamically tracks the ball
* ⚡ Realistic physics with **ball spin mechanics**
* 🧮 **Score tracking** for player and AI
* 💥 Accurate **collision detection** (walls + paddles)
* 🎨 Modern UI with **cyan glowing effects**
* ⏯️ **Pause/Resume functionality** using `SPACE`
* 📱 Responsive gameplay experience

---

## 🧠 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & animations
* **JavaScript (Vanilla JS)** – Game logic & physics

---

## 🎯 Controls

| Action             | Control            |
| ------------------ | ------------------ |
| Move Paddle        | Mouse / Arrow Keys |
| Start / Pause Game | `SPACE` key        |

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/pong-game.git
```

2. Navigate to the project folder:

```bash
cd pong-game
```

3. Open `index.html` in your browser
   *(or use Live Server in VS Code)*

---

## 🧩 Game Logic Highlights

* Ball reflection angle changes based on **collision point**
* AI paddle follows ball with controlled speed (not perfect → playable)
* Frame-based rendering using `requestAnimationFrame`
* Physics simulation for smooth gameplay
